# ✦ AETHER TAC TOE | ETERNAL DUEL ✦

A futuristic and visually immersive Tic Tac Toe game built with React, HTML, CSS, and JavaScript.

AETHER TAC TOE transforms the classic Tic Tac Toe experience into a premium battle arena with animated effects, glowing UI, AI battles, sound effects, score tracking, result screens, multiple board sizes, and dark/light themes.

---

## 🚀 Features

### 🎮 Game Modes

* Friend Mode for 2-player local battles
* AI Mode with difficulty levels:

  * Easy
  * Medium
  * Hard

### 🧠 Smart AI

* Easy AI makes random moves
* Medium AI can win and block strategically
* Hard AI prioritizes:

  * Winning moves
  * Blocking player wins
  * Taking center positions
  * Taking corners

### 🌟 Premium UI Experience

* Dark and Light Theme Toggle
* Glassmorphism panels
* Animated glowing orbs background
* Smooth transitions and hover effects
* Fully responsive design for desktop, tablet, and mobile
* Futuristic fonts using Orbitron, Inter, and Space Grotesk

### 🏆 Result Screen

* Full-screen animated victory / defeat / draw modal
* Custom messages based on game outcome
* Confetti celebration for victories
* Score tracking for:

  * Player X
  * Player O
  * Draws

### 🔊 Sound & Feedback

* Tap sound effect
* Win sound effect
* Lose sound effect
* Draw sound effect
* Optional voice announcements
* Mobile vibration feedback support

### ⚙️ Game Customization

* Choose board sizes:

  * 3x3
  * 4x4
  * 5x5
* Select your symbol in AI Mode:

  * X
  * O
* Custom player names
* Reset game anytime
* Return to menu anytime

### 💾 Persistent Storage

Uses localStorage to save:

* Theme preference
* Score history
